# Number Guessing Game

A fun guessing game where you try to guess a secret number between 0 and 9.

## How to Run
```bash
python number_guessing_game.py
```

## How It Works
1. Computer picks a random number between 0 and 9
2. You guess the number
3. Computer tells you if your guess is too high or too low
4. Keep guessing until you get it right!
5. Game shows how many attempts it took

## Example Gameplay
```
Start game
Guess a number between 0 and 9: 5
Too low
Guess a number between 0 and 9: 7
Too high
Guess a number between 0 and 9: 6
Correct!
Attempts: 3
```

## Features
✅ Random number generation
✅ Input validation (rejects invalid inputs)
✅ Attempt counter
✅ Quit anytime with CTRL+C
