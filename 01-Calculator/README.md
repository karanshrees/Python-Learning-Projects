# Simple Calculator

A basic calculator program that performs simple math operations.

## How to Run
```bash
python simple_calculator.py
```

## What It Does
- Takes two numbers from you
- Asks which operation you want (+, -, *, /)
- Shows you the result

## Example
```
Enter first number: 10
Enter operation (+, -, *, /): +
Enter second number: 5
Result: 15
```
